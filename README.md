# git-2-homeworks-pr 🚀

## 📖 Описание
Проект представляет собой статическую веб-страницу с галереей изображений. Содержит адаптивный дизайн на основе Bootstrap и набор качественных фотографий природы и животных.

## 🛠 Технологический стек
- **HTML5** — разметка страницы
- **CSS3** — стилизация (Bootstrap 4 + кастомные стили)
- **JavaScript** — Bootstrap JS, jQuery, Popper.js
- **Bootstrap 4** — адаптивная сетка и компоненты

## 📦 Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/git-2-homeworks-pr.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd git-2-homeworks-pr
   ```
3. Откройте `index.html` в браузере.

## 💻 Пример использования
Просто откройте `index.html` в любом современном браузере. Страница загрузится с галереей изображений и адаптивным интерфейсом.
