# git-2-homeworks-pr 🚀

## 📖 Описание
Проект представляет собой статическую веб-страницу с галереей изображений. Создан в рамках домашнего задания для отработки навыков работы с Git и GitHub.

## 🛠 Технологический стек
- **HTML5** — разметка страницы
- **CSS3** — стилизация (Bootstrap 4 + кастомные стили)
- **JavaScript** — интерактивность (Bootstrap JS, jQuery, Popper.js)

### Библиотеки и фреймворки
- Bootstrap 4 (CSS + JS)
- jQuery 3 (slim)
- Popper.js

## 📦 Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/git-2-homeworks-pr.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd git-2-homeworks-pr
   ```
3. Откройте `index.html` в браузере.

> **Примечание:** Для корректной работы Bootstrap JS требуется интернет-соединение (CDN) или локальная установка зависимостей.

## 💻 Пример использования
Просто откройте `index.html` в любом современном браузере. Страница содержит галерею изображений с адаптивной версткой.
