# git-2-homeworks-pr 🚀

## 📝 Описание
Проект представляет собой статическую веб-страницу с галереей изображений природы и животных. Реализован с использованием Bootstrap для адаптивной верстки. Подходит для демонстрации навыков работы с HTML, CSS и JavaScript.

## 🛠 Технологический стек
- **HTML5** — разметка страницы
- **CSS3** — стилизация (Bootstrap 4 + кастомные стили)
- **JavaScript** — интерактивность (Bootstrap JS, jQuery, Popper.js)
- **Bootstrap 4** — адаптивная сетка и компоненты

## 📦 Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/git-2-homeworks-pr.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd git-2-homeworks-pr
   ```
3. Откройте `index.html` в браузере:
   - Дважды кликните по файлу, или
   - Используйте Live Server в VS Code

## 💻 Пример использования
```html
<!-- Пример структуры галереи -->
<div class="container">
  <div class="row">
    <div class="col-md-4">
      <img src="img/pexels-drops.jpeg" alt="Капли">
    </div>
    <div class="col-md-4">

    </div>
  </div>
</div>
```
